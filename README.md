# blackjack

> [勉強会用] トランプゲーム

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# firebase command (deploy)
firebase deploy

# login
firebase login

# logout
firebase logout
```
